Cape Arcade

My first UX focused design project.
‍
Cape Arcade was conceptualized as a local Arcadia in the heart of Cape Town. Its loyal patrons are classic game connoisseurs who love the variety on offer, as well as novel gaming experiences. With a dedicated mobile app, Cape Arcade will allow patrons to book both games and private rooms for group events.
For the casuals out there, the app also provides a complete catalogue with plenty of information on what games and experiences are on offer.

My website

A website for showcasing my projects and skills

I started building this website as part of the Google UX design certificate program. The program recommended Squarespace or Wordpress but I wanted to be more hands-on with the development process in order to sharpen my web development skills. The basic static elements were added using Webflow with later additions and changes being added manually.

Spend Tracker

A prototype daily spend tracker
‍
Spend tracker is my solution to a problem I face daily - keeping track of the small things I spend money on. The WatchOS utility app is small and simple by design to allow for quick interactions and tracking. It is the perfect companion app for those who want to take full ownership of how they spend their money.

Brewer

A passion project in progress.

Brewer is a web app for both professional and hobbyist coffee baristas. Initially a small, low code application programmed in Golang, Brewer is now a fully fledged web app for preparing coffee using a variety of coffee brewing techniques and beans.
Future additions will include the functionality for users to book a professional barista for events - with the barista using Brewer to advertise their skills and unique brewing style.
